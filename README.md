# Boreal DS

New Proximus Group Design System

## Prerequisites

This project requires **Node.js 22.21.1** or higher.

### Node.js Version Manager (Recommended)

We recommend using [fnm (Fast Node Manager)](https://github.com/Schniz/fnm) for automatic Node.js version switching. fnm will automatically use the correct Node.js version when you enter the project directory.

For installation and setup instructions, see the [official fnm documentation](https://github.com/Schniz/fnm#installation).

## Installation

1. **Install Node.js 22.21.1:**
   ```bash
   fnm install
   ```

   This will automatically install the Node.js version specified in `.nvmrc`.

2. **Verify Node.js version:**
   ```bash
   node --version
   # Should output: v22.21.1
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```
