<Subtitle>Typography Styles</Subtitle>
  
  Boreal defines a set of typography styles that combine font size, weight, and line height to create a consistent type system across all interfaces. These styles are organized into three main categories: Heading, Body, and Supporting text.

  <h3>How to use it?</h3>
  1. Ensure the style guidelines are installed in the project and loaded at the root or component level.
  ```ts
  npm i @telesign/boreal-style-guidelines
  ```
  2. Import the styles in the root or target element.
  * **SCSS or JS**
  ```scss
  @import "@telesign/boreal-style-guidelines/scss/helpers";
  ```
  * **CSS**
  ```css
    @import url("@telesign/boreal-style-guidelines/scss/helpers");
  ```
  3. Apply any of the following styles to your element.
  ```html
    <p class="display-lg">This is a large body text style.</p>
  ```
