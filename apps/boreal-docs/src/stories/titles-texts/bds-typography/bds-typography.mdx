The Boreal Typography component provides consistent text styling across the application, with multiple variants, sizes, and states to match different design contexts.

<Subtitle>How to use it</Subtitle>

1. Import the component into your project.

```ts
import { BdsTypography } from '@telesign/boreal-web-components';
```

2. Use the `<bds-typography>` tag to display the component in your HTML.

```html
<bds-typography variant="heading" element="h1">Title</bds-typography>
```

<Subtitle>When to use it</Subtitle>

The Typography component provides consistent text styling for various contexts:

    - **Headings** - For page titles, section headers, and content organization
    - **Subheading** - For section subtitles or dividing content groups
    - **Display** - For large, prominent text elements that need visual emphasis
    - **Labels** - For form controls, with support for required and disabled states
    - **Helper** - For providing additional context or assistance, including error messages
    - **Links** - For interactive text elements that navigate to new content
    - **Caption** - For smaller supporting text like image captions or secondary information
    - **Code** - For displaying inline code snippets or technical content
